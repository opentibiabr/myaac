## Instructions

```
1 - Run install.sql file in your canary server database;

2 - Configure pagseguro config on 'plugins\pagseguro\config.php';

3 - Copy and configure files on suggested path:
    get_boxes.lua -> canary\data-otservbr-global\scripts\actions\custom\get_boxes.lua
    receive_items.lua -> canary\data-otservbr-global\scripts\creaturescripts\customs\receive_items.lua

4 - Enjoy the system!  :)
```
